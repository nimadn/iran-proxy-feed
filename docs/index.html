Iran Proxy Feed

Proxies are imported from ProxyScrape when either country = Iran or country_code = IR.

A proxy is published only when it successfully:

  1. Accepts a TCP connection.
  2. Opens ISNA through the proxy.
  3. Opens Google's generate_204 endpoint through the proxy.

A previously successful proxy is retained for up to 30 days after its latest successful test.

  • Full sing-box configuration
    iran-all.json
  • Base64 mixed subscription
    iran-v2ray.txt
  • Readable subscription
    iran-v2ray-plain.txt
  • Detailed test evidence
    iran-test-results.json
  • Generation report
    iran-report.json
  • Persistent 30-day history
    proxy-history.json
Public proxies may inspect, modify, or log traffic. Do not use these proxies for banking, email, passwords, private accounts, or sensitive data.
